# Psy Vibe

Сайт психолога Елизаветы Шихалевой.

Цель сайта:
- презентовать психолога и услуги
- собирать заявки
- вести блог

Стек:
- Astro
- Tailwind CSS
- Keystatic CMS
- статическая сборка
- NetAngels

Страницы:
- Главная
- Обо мне
- Тарифы
- Блог
- Статья блога
- Контакты
- Thank you

Контент будет редактировать нетехнический человек через CMS.

Важно:
- стиль спокойный, аккуратный, премиально-мягкий
- без визуального шума
- структура понятная
- адаптив обязателен
- главная может быть структурно фиксированной, но с редактируемыми текстами, кнопками, картинками и карточками
- блог простой, без категорий и тегов
- форма нужна на странице Контакты
